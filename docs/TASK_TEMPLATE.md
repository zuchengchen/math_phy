# 单次任务模板

下面是一份可复制的任务模板。每次请直接在对话中填写并提交，不需要专门再改文件。

```text
请按照本项目讲义规范，编写《数学物理方法》第 X 章第 X 节。

本节主题：
……

本节重点希望讲清：
……

要求：
1. 面向基础较弱的本科生，必须自学友好。
2. 先讲物理背景和问题来源，再讲定义、定理、证明和方法。
3. 语言口语化，像老师上课，不要写成提纲式教材。
4. 推导尽量不跳步，解释每一步为什么这样做。
5. 尽量加入 1-2 个 TikZ 图，帮助建立直观。
6. 术语默认以中文表述，不必强行补中英对照；若确有必要，可少量补充英文。
7. 必须包含：
   - 导入
   - 定义/定理
   - 证明
   - 例题与完整解答
   - 小结
   - 分层习题
8. 输出为可直接放入章节文件的 LaTeX 内容，不要额外解释。
```

## 常见任务变体

### 1. 润色已有内容

```text
请按照本项目讲义规范，润色下面这段 LaTeX 内容。

目标：
1. 更适合基础较弱学生自学
2. 更口语化、更像老师讲课
3. 补清来龙去脉和关键台阶
4. 必要时补充 `tishi`、`jinshi` 或图示建议
5. 术语默认以中文表述；只有在确有必要时再补英文

要求输出：
- 直接给出修改后的 LaTeX
```

### 2. 补图

```text
请为下面这一节补 1-2 个 TikZ 图。

要求：
1. 图必须服务理解，不是装饰
2. 图中文字少而准
3. 版式简洁，适合教材排版
4. 图注和正文默认以中文表述；若确有必要，再局部补英文
5. 输出为可直接插入章节文件的 LaTeX figure 环境
```

### 3. 补例题与习题

```text
请为下面这一节补充：
1. 1 个完整例题（含 `jie` 解答）
2. 基础题、综合题、挑战题各 2 题

要求：
- 题目应和本节核心知识直接相关
- 难度递进清楚
- 例题解答不跳关键步
- 例题与解答默认以中文表述；若确有必要，再补英文术语
```

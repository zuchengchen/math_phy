# 数学物理方法讲义

本项目是数学物理方法课程的 LaTeX 讲义，涵盖复变函数、积分变换、数理方程和特殊函数四大部分。

## 内容概览

### 第一篇：复变函数（第1-6章）
- 复数与复平面
- 复变函数
- 复积分
- 级数表示
- 留数理论
- 保角映射

### 第二篇：积分变换（第7-8章）
- 傅里叶变换
- 拉普拉斯变换

### 第三篇：数理方程（第9-14章）
- 方程导出与分类
- 行波法
- 分离变量法
- 积分变换法
- 格林函数法
- 其他方法

### 第四篇：特殊函数（第15-17章）
- 勒让德函数
- 贝塞尔函数
- 其他特殊函数简介

## 编译方法

使用 XeLaTeX 编译（推荐）：

```bash
xelatex main.tex
xelatex main.tex  # 运行两次以生成正确的目录
```

或使用 LuaLaTeX：

```bash
lualatex main.tex
lualatex main.tex
```

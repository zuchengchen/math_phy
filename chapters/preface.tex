\section*{为什么学这门课？}

同学们好！欢迎来到《数学物理方法》的课堂。

很多同学第一次听到这门课的名字，会觉得它有点``硬''，甚至有点吓人：数学、物理、方法，这三个词一摆在一起，好像就意味着公式很多、计算很长、内容很难。

但我们不妨先不看书名，先看一个很具体的问题：\textbf{为什么吉他琴弦能发出美妙的声音？}
一根弦、两端固定、轻轻拨动——这个看似简单的动作背后，隐藏着深刻的数学。为什么能发出不同音高的声音？为什么声音会随时间衰减？为什么有些振动模式稳定、有些却很快消失？如果你想把这些问题真正想明白，这门课就是为你准备的。

其实，在普通物理和电磁学中，你已经遇到过各种各样的方程：波动方程、热传导方程、拉普拉斯方程……但当时你往往只是``会用''这些方程，还没有系统追问：这些方程是怎么来的？它们为什么这样写？在不同边界条件下又该怎么解？

这门课要做的，就是把这些零散出现的工具和方法一点点串起来，让你逐步学会\textbf{从物理问题出发，建立数学模型，完成数学求解，再回到物理解释}。
\section*{这门课学什么？}

如果把物理专业的课程体系看成一条学习链，那么这门课正好卡在中间很关键的位置：前面连着高等数学和普通物理，后面连着理论力学、电动力学、量子力学、热力学与统计物理。因此，它常被看作一门``承上启下''的桥梁课程。

\noindent
换句话说，这门课不是孤零零多出来的一门课，而是把你前面学过的知识重新组织起来，再送到后面的核心课程里去。

\textbf{前置课程}提供了必要的数学基础和物理直觉：
\begin{itemize}
    \item \textbf{高等数学}：微积分、级数、常微分方程——这是本课程的数学基础。
    \item \textbf{普通物理}（力学、热学、光学、电磁学）：提供了丰富的应用背景。振动、波动、热传导、电磁场等问题，都会在本课程中被更系统地处理。
\end{itemize}

\textbf{后续课程}会频繁用到这里学到的方法：
\begin{itemize}
    \item \textbf{理论力学}：拉格朗日方程、哈密顿方程的求解需要微分方程和变分法。
    \item \textbf{电动力学}：麦克斯韦方程组的求解离不开数学物理方程。
    \item \textbf{量子力学}：薛定谔方程的求解用到分离变量法、特殊函数。
    \item \textbf{热力学与统计物理}：输运方程、玻尔兹曼方程的求解也需要本课程的方法。
\end{itemize}

本课程分为四个部分，它们不是孤立的，而是相互关联的：

\begin{enumerate}
    \item \textbf{复变函数}：把函数的定义域从实数扩展到复数。你会发现，复数不只是``实数的推广''，它有独特的优美结构。复变函数这一部分会帮助你理解解析函数、级数展开和复积分，也会为后面的留数方法打下基础；在二维势场问题中，它还能提供很有启发性的直观图像。
    
    \item \textbf{积分变换}：把一个函数变成另一个函数，看起来像是在``变魔术''，实际上是把微分方程变成代数方程，把复杂问题变简单。傅里叶变换和拉普拉斯变换是解偏微分方程的两把``钥匙''。
    
    \item \textbf{数理方程}：这是核心部分。学习如何从物理问题导出偏微分方程，以及求解这些方程的各种方法——分离变量法、行波法、积分变换法、格林函数法等。前面学到的复变函数和积分变换，在这里大显身手。
    
    \item \textbf{特殊函数}：在解数理方程的过程中，会``自然地''遇到一些特殊的函数——勒让德多项式、贝塞尔函数等。它们不是人为发明的，而是物理问题本身``选择''的函数。球对称问题引出勒让德函数，圆柱对称问题引出贝塞尔函数。
\end{enumerate}

\medskip

如果这门课学扎实了，学完以后你至少应该具备下面几方面能力：

\begin{itemize}
    \item 推导和求解经典的偏微分方程（热传导方程、波动方程、拉普拉斯方程等）
    \item 熟练处理各种边界条件和初始条件
    \item 理解复变函数在二维势场与复积分计算中的基本作用，并掌握留数法计算一些复杂实积分
    \item 运用特殊函数求解球对称、圆柱对称问题（勒让德函数、贝塞尔函数）
    \item 为学习四大力学打下坚实的数学基础
    \item 遇到新的物理问题时，知道``该用什么数学工具''去解决
\end{itemize}

更重要的是，你会逐渐熟悉一种典型的理论物理思路：\textbf{物理问题 $\rightarrow$ 数学模型 $\rightarrow$ 数学求解 $\rightarrow$ 物理解释}。这不仅适用于本课程，也会贯穿你后面的专业学习。

\section*{怎么学这门课？}

先把话说在前面：这门课确实不算轻松。它的难点不只在公式多、计算长，更在于你需要同时照顾物理背景、数学推导和结果解释。初学时，下面几类问题最容易卡住：

\begin{itemize}
    \item \textbf{复变函数}：从实数到复数的思维转换需要时间适应。复变函数的``导数''定义看起来和实函数一样，但条件苛刻得多。学习时要多画图，在复平面上理解几何意义，并把实变函数的结论和复变函数对照着看。
    
    \item \textbf{积分变换}：傅里叶变换和拉普拉斯变换的公式多，容易记混，收敛域又常常被忽略。学习时不要只背公式，要抓住它的本质：把微分运算变成乘法，再自己推几遍核心公式，把它们之间的联系理顺。
    
    \item \textbf{分离变量法}：公式本身不复杂，真正的难点是边界条件。什么样的边界条件能用分离变量，边界条件又该怎样代入，往往决定整道题能不能做下去。练习时最好从齐次边界条件起步，并养成先画边界示意图的习惯。
    
    \item \textbf{特殊函数}：名称多、性质多，刚开始很容易``晕头转向''。这部分不要靠死记硬背顶过去，而要从它们的来源和物理意义入手，先弄清楚球对称问题为什么引出勒让德函数，圆柱对称问题为什么引出贝塞尔函数。
    
    \item \textbf{推导过程长}：很多定理证明和例题计算步骤多，最怕一路往下抄却不知道自己在做什么。读长推导时，不妨强迫自己在关键地方停下来，问一句``为什么这一步可以这样做''，把逻辑线抓住以后，后面的计算就不容易跟丢。
\end{itemize}

不过你也不用一开始就被它吓住。这门课有一个很鲜明的特点：前面的工具会在后面反复出现。只要把几个核心方法真正弄懂，后面的内容就会越来越顺。

\begin{itemize}
    \item \textbf{先问物理问题是什么}：每个数学概念都不是凭空出现的。先搞清楚它到底想解决什么问题，再去看公式和定理，理解会稳很多。
    
    \item \textbf{把前修知识随时接上}：积分技巧、级数展开、常微分方程、向量分析这些内容都会反复用到。遇到卡点时，及时回头补，不要硬撑着往后赶。
    
    \item \textbf{坚持动手算}：数学物理方法是``算''出来的，不是``看''出来的。很多步骤看上去明白了，真正下笔时问题才会暴露出来。
    
    \item \textbf{格外重视细节}：收敛域、边界条件、奇点位置、展开区间这些地方，往往决定一整道题能不能做对。
    
    \item \textbf{主动建立联系}：复变函数和二维势场、傅里叶变换和分离变量法、特殊函数和对称性，这些内容不是分开的。把联系看清楚，学习效率会高很多。
\end{itemize}

\begin{tishi}[动手算，才能真正掌握]
数学物理方法是``算''出来的，不是``看''出来的。看懂了不等于会做了。建议：每道例题先自己算一遍，再看答案。算错了没关系，重要的是找出错误原因——这往往比算对收获更大。
\end{tishi}


\section*{这份讲义怎么帮你学？}

上课时，老师会提醒你哪里容易出错，哪里要和前面的知识联系起来，哪里又该从物理图像去理解。这份讲义也尽量扮演这样的角色。

讲义中的\textcolor{red!70!black}{红色警示框}标注易错点，\textcolor{blue}{蓝色提示框}提供学习建议，\textcolor{green!50!black}{绿色背景框}补充物理背景。阅读时留意这些``路标''，往往能少走不少弯路。

这份讲义从一开始就有一个很明确的目标：\textbf{尽量写成一份适合学生自学的讲义}。它不仅希望你在课堂上能跟着老师走，也希望你在课后一个人翻开它时，依然知道每一步在讲什么、为什么要这样讲、前后内容又是怎么连起来的。

编写时，我始终默认读者的基础未必特别扎实，但愿意认真学、愿意一步一步往前走。因此，很多地方会故意讲得慢一点、细一点，把关键台阶交代清楚，而不是只把结论摆在你面前。

具体来说，这份讲义在写法上会尽量做到下面几点：

\begin{itemize}
    \item \textbf{先讲来由，再讲结论}：抽象概念尽量先从物理问题、几何图像或典型例子引入，让你先知道``为什么会出现这个东西''，再去看正式定义和结论。
    
    \item \textbf{把来龙去脉说清楚}：重要公式、定理和方法会尽量把推导过程写完整，把中间真正关键的环节解释清楚，让你知道每一步为什么成立，而不是只记住最后一个结果。
    
    \item \textbf{尽量用口语化的方式讲清楚}：我希望它读起来更像老师在黑板前一步步讲，而不是像一本只给结论的手册。能直说的地方尽量不绕，能用直观语言说明的地方尽量不用过分生硬的表述。
    
    \item \textbf{在需要的地方补前修知识和图像}：遇到与高等数学、普通物理、常微分方程相关的内容，会补充必要的提醒、回顾和解释；复平面、积分路径、边界条件、振动模式等抽象内容，也会尽量配上示意图，帮助你把公式和图像对应起来。
    
    \item \textbf{帮助你把知识串起来}：不仅介绍单个结论，也尽量提示不同章节之间的联系，让你看到一门课的整体脉络，而不是把它学成一堆零散技巧。
    
    \item \textbf{让复习也有抓手}：每章都安排了例题、小结和分层习题，方便你学完以后及时巩固，也方便考前回头整理思路。
\end{itemize}

如果你在阅读这份讲义时，能感觉它不像在``啃一本教材''，而更像有一位老师在旁边陪着你，把概念慢慢讲开，把难点一层层拆开，那么这份讲义就达到它的目的了。

% 第三章：复积分与柯西理论

\chapter{复积分与柯西理论}

\begin{wulibj}[本章导读]
第二章里，我们已经知道：解析函数不是普通的``二维函数''，它满足柯西--黎曼条件，和调和函数、二维势场、局部保角性都有紧密联系。接下来，一个非常自然的问题就是：如果我们把这样的函数沿复平面中的曲线积分，会发生什么？

一开始，复积分看上去只是把实变积分里的 $\mathrm{d}x$ 换成了 $\mathrm{d}z$。但真正学下去以后，你会发现差别远不止于此。对于一般连续函数，积分当然会依赖路径；可对于解析函数，情况会突然变得异常整齐：闭路积分往往为零，积分可能只由端点决定，甚至区域内部一点的函数值都能由边界上的积分表示出来。

本章要完成下面几件事：
\begin{enumerate}
    \item 建立复积分的定义、参数化计算方法和基本性质；
    \item 理解路径无关、原函数与闭路积分之间的关系；
    \item 掌握柯西积分定理\index{柯西积分定理}与柯西积分公式这两条主线定理；
    \item 为下一章的泰勒展开、解析函数刚性以及第五章的留数理论作准备。
\end{enumerate}

如果把本章压缩成一条主线，那就是
\[
    \begin{aligned}
        \text{解析函数}
        &\Longrightarrow \text{闭路积分为零}
        \Longrightarrow \text{路径无关与原函数} \\
        &\Longrightarrow \text{柯西积分公式}
        \Longrightarrow \text{解析函数的刚性}.
    \end{aligned}
\]
请大家带着这个主线往下看。这样后面遇到各种公式时，就不容易把它们看成彼此孤立的技巧。
\end{wulibj}

% ========== 第一节 ==========
\section{复积分的定义、参数化与基本性质}

\begin{wulibj}[为什么会出现沿曲线的积分]
在实变函数里，我们最熟悉的是沿区间积分，因为自变量只在一条数轴上变化。但到了复平面里，很多对象天然就是沿曲线或边界出现的：例如二维势场中的边界环流、沿闭合边界累加场的贡献，以及后面要看到的``边界决定内部''现象，都要求我们把积分对象放到一条有向曲线上去看。

这时有一个很重要的变化：我们不只关心曲线经过了哪些点，还关心它是按什么方向走过去的。也正因为如此，复积分里出现的是 $\Delta z$ 或 $\mathrm{d}z$，而不是单纯的弧长元 $\mathrm{d}s$。前者保留了方向信息，后者只保留长度信息；这也是复积分会表现出比普通弧长积分更丰富结构的原因。
\end{wulibj}

\subsection{复积分的定义}

\begin{dingliyi}{复积分}{}
设 $C$ 是复平面上一条有向光滑曲线，起点为 $A$，终点为 $B$。函数 $f(z)$ 在 $C$ 上有定义。将 $C$ 任意分割成 $n$ 段，分点为 $z_0 = A, z_1, \ldots, z_n = B$。在每个小弧段上任意取点 $\zeta_k$，作和
\begin{equation}
    S_n = \sum_{k=1}^{n} f(\zeta_k)(z_k - z_{k-1}) = \sum_{k=1}^{n} f(\zeta_k)\Delta z_k
\end{equation}
若当 $\max|\Delta z_k| \to 0$ 时，$S_n$ 的极限存在且与分割方式和 $\zeta_k$ 的取法无关，则称此极限为 $f(z)$ 沿曲线 $C$ 的\textbf{复积分}，记作
\begin{equation}
    \int_C f(z)\,\mathrm{d}z = \lim_{\max|\Delta z_k| \to 0} \sum_{k=1}^{n} f(\zeta_k)\Delta z_k
\end{equation}
\end{dingliyi}

\begin{zhu}{}{}
这里最容易忽略的一点是：和式里出现的是 $\Delta z_k$，而不是 $|\Delta z_k|$。这不是记号上的小差别，而是定义本身的关键。若写成 $|\Delta z_k|$，记录的只是小弧段的长度；而写成 $\Delta z_k$，记录的则是复平面中的位移，因此方向信息也被一起保留下来了。后面闭路积分会不会为零、改方向后为什么会变号，都和这件事直接相关。
\end{zhu}

\subsection{复积分的参数化与计算}

真正计算复积分时，最常用的入口不是直接回到黎曼和定义，而是先把曲线参数化。设
\[
    z=z(t),\qquad a\le t\le b,
\]
其中 $z(a)$ 是曲线起点，$z(b)$ 是曲线终点。则
\[
    \mathrm{d}z = z'(t)\,\mathrm{d}t,
\]
从而
\[
    \int_C f(z)\,\mathrm{d}z = \int_a^b f(z(t))z'(t)\,\mathrm{d}t.
\]
这条公式非常重要，因为它把``沿复平面中的曲线积分''改写成了熟悉的实变量积分。后面不论是直接计算，还是证明各种性质，几乎都要从这里出发。

设 $f(z) = u(x, y) + \mathrm{i}v(x, y)$，$z = x + \mathrm{i}y$，$\mathrm{d}z = \mathrm{d}x + \mathrm{i}\mathrm{d}y$。则
\begin{align}
    \int_C f(z)\,\mathrm{d}z &= \int_C (u + \mathrm{i}v)(\mathrm{d}x + \mathrm{i}\mathrm{d}y) \\
    &= \int_C (u\,\mathrm{d}x - v\,\mathrm{d}y) + \mathrm{i}\int_C (v\,\mathrm{d}x + u\,\mathrm{d}y)
\end{align}

这把复积分化为两个实线积分。

图~\ref{fig:parametrization-geometry} 想强调的是：参数化并不是把曲线``写成一个公式''这么简单，而是把沿曲线积分这件事改写成``随着参数 $t$ 前进而累加贡献''。这样一来，复积分就可以回到熟悉的实变量积分框架里去计算。

\begin{lizi}{}{}
计算 $\displaystyle\int_C z\,\mathrm{d}z$，其中 $C$ 是从 $0$ 到 $1 + \mathrm{i}$ 的直线段。

\begin{jie}
    参数化曲线：$z(t) = t(1 + \mathrm{i})$，$t \in [0, 1]$。
    
    则 $\mathrm{d}z = (1 + \mathrm{i})\,\mathrm{d}t$，$z = t(1 + \mathrm{i})$。
    
    \begin{align}
        \int_C z\,\mathrm{d}z &= \int_0^1 t(1 + \mathrm{i}) \cdot (1 + \mathrm{i})\,\mathrm{d}t \\
        &= (1 + \mathrm{i})^2 \int_0^1 t\,\mathrm{d}t \\
        &= 2\mathrm{i} \cdot \frac{1}{2} = \mathrm{i}
    \end{align}
\end{jie}
\end{lizi}

\subsection{复积分的基本性质}

\begin{dingli}{复积分的基本性质}{}
设 $f(z)$ 和 $g(z)$ 沿曲线 $C$ 可积，则：
\begin{enumerate}
    \item $\displaystyle\int_C [f(z) + g(z)]\,\mathrm{d}z = \int_C f(z)\,\mathrm{d}z + \int_C g(z)\,\mathrm{d}z$
    \item $\displaystyle\int_C kf(z)\,\mathrm{d}z = k\int_C f(z)\,\mathrm{d}z$（$k$ 为常数）
    \item $\displaystyle\int_{C_1 + C_2} f(z)\,\mathrm{d}z = \int_{C_1} f(z)\,\mathrm{d}z + \int_{C_2} f(z)\,\mathrm{d}z$
    \item $\displaystyle\int_{C^-} f(z)\,\mathrm{d}z = -\int_C f(z)\,\mathrm{d}z$（$C^-$ 是 $C$ 的反向曲线）
    \item $\displaystyle\left|\int_C f(z)\,\mathrm{d}z\right| \leq \int_C |f(z)|\,|\mathrm{d}z| = \int_C |f(z)|\,\mathrm{d}s$
\end{enumerate}
\end{dingli}

\begin{proof}
前四条性质都可以直接从复积分的定义出发验证，因为黎曼和本身就满足线性、分段相加和反向取负这些代数性质。这里把最需要单独说明的估值不等式写出来。

设曲线 $C$ 的参数方程为
\[
    z=z(t),\qquad a\le t\le b.
\]
则
\[
    \int_C f(z)\,\mathrm{d}z = \int_a^b f(z(t))z'(t)\,\mathrm{d}t.
\]
由三角不等式，
\[
    \left|\int_C f(z)\,\mathrm{d}z\right|
    \leq \int_a^b |f(z(t))|\,|z'(t)|\,\mathrm{d}t.
\]
而
\[
    \mathrm{d}s = |z'(t)|\,\mathrm{d}t,
\]
所以
\[
    \left|\int_C f(z)\,\mathrm{d}z\right|
    \leq \int_C |f(z)|\,\mathrm{d}s.
\]
这就证明了性质5。
\end{proof}

\begin{zhu}{}{}
性质5称为\textbf{积分估值不等式}，其中 $\mathrm{d}s$ 是弧长微元。它给出积分模的上界。
\end{zhu}

\begin{tishi}[这些性质为什么重要，做题时怎么用]
这些基本性质看起来朴素，但后面几乎每一页都要用到。线性性质告诉我们：复杂积分常常可以拆成几项分别处理；曲线可分段相加告诉我们：路径若由若干段拼起来，可以一段一段地算；反向变号告诉我们：一旦改了方向，符号必须跟着改；而积分估值不等式则是证明中的常客，尤其在让某个小圆弧积分趋于零、或者控制误差项时，往往第一反应就应该想到它。
\end{tishi}

% ========== 第二节 ==========
\section{路径无关、原函数与复积分基本公式}

\begin{wulibj}[为什么这里先谈路径无关]
上一节我们已经看到，复积分的定义本身是沿曲线作和，因此对一般函数来说，积分自然会依赖路径。可一旦某个函数存在原函数，积分就会突然变得和实变积分很像：它只和起点、终点有关，而与中间具体怎样走无关。

如果你学过普通物理里``保守力做功与路径无关''这件事，就会觉得这里很眼熟：做功只由始末位置决定，本质上正是因为存在势函数。复积分里的``原函数''扮演的就是类似角色。

这一节先把这种``如果有原函数，就会路径无关''的思路讲清楚。下一节再用柯西积分定理说明：为什么在单连通区域里，解析函数恰好会拥有这种特殊性质。
\end{wulibj}

\subsection{什么叫积分与路径无关}

设 $z_1,z_2$ 是区域 $D$ 内两点，$C_1,C_2$ 是连接 $z_1$ 与 $z_2$ 的两条分段光滑曲线。若对任意这样的两条曲线都有
\[
    \int_{C_1} f(z)\,\mathrm{d}z = \int_{C_2} f(z)\,\mathrm{d}z,
\]
则称 $f(z)$ 在区域 $D$ 内的积分\textbf{与路径无关}。

\begin{zhu}{}{}
``与路径无关''不是一个定义性的空话，而是一个很强的性质。它的意思是：从同一点出发走到同一点终止时，中间绕不绕弯、沿哪条路走，都不会改变积分值。
\end{zhu}

\subsection{原函数与复积分的基本公式}

\begin{dingliyi}{原函数}{}
设 $f(z)$ 在区域 $D$ 内连续。若存在 $F(z)$ 使得
\[
    F'(z)=f(z)
\]
在 $D$ 内成立，则称 $F(z)$ 是 $f(z)$ 的一个\textbf{原函数}。
\end{dingliyi}

\begin{dingli}{有原函数时的复积分基本公式}{}
\index{牛顿@牛顿（Newton）}\index{莱布尼茨@莱布尼茨（Leibniz）}
设 $f(z)$ 在区域 $D$ 内有原函数 $F(z)$。若 $C$ 是 $D$ 内一条从 $z_1$ 到 $z_2$ 的分段光滑曲线，则
\[
    \int_C f(z)\,\mathrm{d}z = F(z_2)-F(z_1).
\]
\end{dingli}

\begin{proof}
沿曲线 $C$ 作参数化
\[
    z=z(t),\qquad a\le t\le b,
\]
其中
\[
    z(a)=z_1,\qquad z(b)=z_2.
\]
于是
\[
    \int_C f(z)\,\mathrm{d}z
    = \int_a^b f(z(t))z'(t)\,\mathrm{d}t
    = \int_a^b F'(z(t))z'(t)\,\mathrm{d}t.
\]
由链式法则，
\[
    \frac{\mathrm{d}}{\mathrm{d}t}F(z(t)) = F'(z(t))z'(t),
\]
因此
\[
    \int_C f(z)\,\mathrm{d}z
    = \int_a^b \frac{\mathrm{d}}{\mathrm{d}t}F(z(t))\,\mathrm{d}t
    = F(z(b))-F(z(a))
    = F(z_2)-F(z_1).
\]
这就得到了所需公式。
\end{proof}

\begin{tishi}[这条公式的意义和用法]
这条公式就是复积分里的牛顿--莱布尼茨公式。它真正告诉我们的不是``又多了一个公式''，而是：只要原函数存在，积分就不再依赖中间具体怎样走，而只看起点和终点。

因此做题时，若被积函数一眼就能看出是某个函数的导数，例如 $f(z)=z^n$、$\mathrm{e}^z$、$\sin z$、$\cos z$ 等，或者题目已经告诉你原函数存在，就不要再老老实实参数化积分，而应先试着直接写出原函数。
\end{tishi}

\begin{tuilun}{有原函数时积分与路径无关}{}
若 $f(z)$ 在区域 $D$ 内有原函数，则它在 $D$ 内的积分与路径无关；特别地，对任意闭曲线 $C$，都有
\[
    \oint_C f(z)\,\mathrm{d}z = 0.
\]
\end{tuilun}

\begin{proof}
若 $C_1,C_2$ 都从 $z_1$ 走到 $z_2$，则由上一定理，
\[
    \int_{C_1} f(z)\,\mathrm{d}z = F(z_2)-F(z_1)=\int_{C_2} f(z)\,\mathrm{d}z.
\]
所以积分与路径无关。

当 $C$ 是闭曲线时，起点与终点重合，于是
\[
    \oint_C f(z)\,\mathrm{d}z = F(z_1)-F(z_1)=0.
\]
\end{proof}

\begin{tishi}[这条推论最常怎么用]
如果题目问``两条不同路径上的积分是否相等''，或者直接问``某个闭路积分是不是零''，先不要急着算。更高效的判断往往是：这个函数有没有原函数？如果有，那么这两类问题就可以一步结束。
\end{tishi}

\begin{tishi}[这一节和下一节的关系]
到目前为止，我们只是说明了：\textbf{如果}原函数存在，那么积分就会路径无关。下一节真正关键的内容是：为什么单连通区域里的解析函数会满足闭路积分为零，从而反过来获得原函数。这正是柯西积分定理\index{柯西积分定理}的威力所在。
\end{tishi}

% ========== 第三节 ==========
\section{柯西积分定理与闭路积分\index{柯西积分定理}}

\subsection{定理的陈述}

\begin{dingli}{柯西积分定理}{}
设 $f(z)$ 在单连通区域 $D$ 内解析，$C$ 是 $D$ 内任意一条简单闭曲线，则
\[
    \oint_C f(z)\,\mathrm{d}z = 0
\]
\end{dingli}

\begin{tishi}[这条定理为什么是全章枢纽]
柯西积分定理真正厉害的地方在于：它把``解析''这种局部可导性质，直接转化成了``任意闭路积分为零''这种整体结论。于是原函数、路径无关、柯西积分公式，都会顺着这条定理往后长出来。

做题时，只要看到\textbf{闭曲线积分}，就要立刻检查三件事：曲线是否闭合；函数是否在曲线及其内部解析；区域里是否没有洞。只有这些条件都满足时，才能直接把积分判为零。
\end{tishi}

\subsection{定理的证明（格林公式方法）}

\begin{proof}
记 $\Omega$ 为简单闭曲线 $C$ 所围成的区域。由于 $D$ 是单连通区域，而且 $C$ 完全落在 $D$ 内，所以 $C$ 的内部区域 $\Omega$ 也全部包含在 $D$ 中；这正是单连通条件在这里发挥作用的地方。下面先默认 $C$ 取正向（逆时针方向）；若取反向，积分只差一个负号，而零乘以 $-1$ 仍然是零。

把 $f$ 写成
\[
    f(z)=u(x,y)+\mathrm{i}v(x,y).
\]
则
\begin{align}
    \oint_C f(z)\,\mathrm{d}z
    &= \oint_C (u+\mathrm{i}v)(\mathrm{d}x+\mathrm{i}\,\mathrm{d}y) \\
    &= \oint_C (u\,\mathrm{d}x-v\,\mathrm{d}y)
    + \mathrm{i}\oint_C (v\,\mathrm{d}x+u\,\mathrm{d}y).
\end{align}

现在把实部和虚部分开处理。对第一个积分，在格林公式里取
\[
    P=u,\qquad Q=-v,
\]
便得到
\begin{align}
    \oint_C (u\,\mathrm{d}x-v\,\mathrm{d}y)
    &= \iint_{\Omega}\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)\,\mathrm{d}x\mathrm{d}y \\
    &= \iint_{\Omega}\left(-\frac{\partial v}{\partial x}-\frac{\partial u}{\partial y}\right)\,\mathrm{d}x\mathrm{d}y.
\end{align}

对第二个积分，在格林公式里取
\[
    P=v,\qquad Q=u,
\]
便得到
\[
    \oint_C (v\,\mathrm{d}x+u\,\mathrm{d}y)
    = \iint_{\Omega}\left(\frac{\partial u}{\partial x}-\frac{\partial v}{\partial y}\right)\,\mathrm{d}x\mathrm{d}y.
\]

由于 $f$ 在 $D$ 内解析，在本章采用的证明框架下，$u,v$ 满足柯西--黎曼条件
\[
    u_x=v_y,\qquad u_y=-v_x.
\]
于是
\[
    -v_x-u_y=0,\qquad u_x-v_y=0.
\]
因此上面两个面积分都等于零，从而
\[
    \oint_C f(z)\,\mathrm{d}z = 0.
\]
\end{proof}

\begin{zhu}{}{}
这个证明要求 $u_x, u_y, v_x, v_y$ 连续。更一般的证明（不要求偏导数连续）需要更细致的分析。
\end{zhu}

\subsection{闭路积分为零带来的后果}

\begin{dingli}{单连通区域中原函数的存在性}{}
若 $f(z)$ 在单连通区域 $D$ 内解析，则 $f(z)$ 在 $D$ 内存在原函数。
\end{dingli}

\begin{proof}
取定一点 $z_0\in D$。对任意 $z\in D$，定义
\[
    F(z)=\int_{z_0}^{z} f(\zeta)\,\mathrm{d}\zeta,
\]
其中积分路径取在区域 $D$ 内。

由柯西积分定理\index{柯西积分定理}，任意闭曲线上的积分都为零，因此两点之间的积分与路径无关，所以 $F(z)$ 的定义是良好的。

现在证明 $F'(z)=f(z)$。固定一点 $z\in D$。由于 $D$ 是开区域，存在充分小的 $h$，使得从 $z$ 到 $z+h$ 的线段仍全部落在 $D$ 内。由路径无关性，
\[
    F(z+h)-F(z)=\int_z^{z+h} f(\zeta)\,\mathrm{d}\zeta.
\]
把这条线段参数化为
\[
    \zeta=z+th,\qquad 0\le t\le 1,
\]
则
\[
    \mathrm{d}\zeta=h\,\mathrm{d}t,
\]
因此
\[
    \frac{F(z+h)-F(z)}{h}
    = \int_0^1 f(z+th)\,\mathrm{d}t.
\]
由于 $f$ 在 $z$ 处连续，对任意 $\varepsilon>0$，存在 $\eta>0$，只要 $|w-z|<\eta$，就有
\[
    |f(w)-f(z)|<\varepsilon.
\]
当 $|h|<\eta$ 时，对所有 $t\in[0,1]$ 都有
\[
    |z+th-z|=t|h|\le |h|<\eta,
\]
所以整条小线段上的函数值都会同时靠近 $f(z)$。于是
\begin{align}
    \left|\int_0^1 f(z+th)\,\mathrm{d}t-\int_0^1 f(z)\,\mathrm{d}t\right|
    &\le \int_0^1 |f(z+th)-f(z)|\,\mathrm{d}t \\
    &\le \int_0^1 \varepsilon\,\mathrm{d}t=\varepsilon.
\end{align}
因此
\[
    \lim_{h\to 0}\frac{F(z+h)-F(z)}{h}
    = \int_0^1 f(z)\,\mathrm{d}t
    = f(z).
\]
因此 $F(z)$ 正是 $f(z)$ 的一个原函数。
\end{proof}

\begin{tishi}[这条存在定理的意义和用法]
这一条定理把前一节的逻辑真正补全了。前一节说的是``如果已经有原函数，会怎样''；这一节则进一步告诉我们：在单连通区域里，只要函数解析，原函数其实自动存在。

所以今后判断积分是否路径无关时，常见的思路就变成了：先检查函数是否在单连通区域里解析；如果是，就可以直接知道它有原函数，从而路径无关、闭路积分为零。
\end{tishi}

\begin{tishi}[三件事其实连在一起]
到这里为止，我们已经看到了下面这条主线：
\[
    \text{闭路积分为零} \Longrightarrow \text{路径无关} \Longrightarrow \text{原函数存在}.
\]
而在单连通区域里，解析函数之所以拥有这些性质，根源正是柯西积分定理。换句话说，复积分之所以``突然变整齐''，并不是偶然，而是解析性带来的结果。
\end{tishi}

% ========== 第四节 ==========
\section{多连通区域与典型闭路积分}

\begin{wulibj}[为什么单连通条件不能随手丢掉]
上一节的柯西积分定理是在单连通区域里成立的。很多同学第一次看到这个条件时，会把它当成一句技术性限制；但一进入多连通区域，情形就会立刻改变。最典型的例子就是
\[
    \frac{1}{z-z_0}.
\]
它除了 $z_0$ 以外处处解析，可一旦曲线把 $z_0$ 围住，闭路积分就不再是零。

从物理直觉上说，这就像区域里出现了一个不能穿过去的障碍物、导体孔洞或被挖去的点。此时你沿边界绕行一圈所累加出来的信息，可能会记住``到底有没有绕住那个洞''，而不再像单连通情形那样可以随意缩成一点。

这一节的任务，就是让大家真正看见：区域里的``洞''会怎样改变积分行为。
\end{wulibj}

\subsection{多连通区域的柯西积分定理\index{柯西积分定理}}

\begin{dingli}{多连通区域的柯西积分定理}{}
设 $f(z)$ 在多连通区域 $D$ 内解析，$C$ 是 $D$ 的外边界，$\Gamma_1,\Gamma_2,\ldots,\Gamma_n$ 是 $D$ 的内边界，并且都按区域的正向取向（外边界逆时针，内边界顺时针）。则
\[
    \oint_C f(z)\,\mathrm{d}z + \sum_{k=1}^{n} \oint_{\Gamma_k} f(z)\,\mathrm{d}z = 0.
\]
等价地，若把同样的内边界改记为逆时针方向的闭曲线 $C_1,C_2,\ldots,C_n$，则
\[
    \oint_C f(z)\,\mathrm{d}z = \sum_{k=1}^{n} \oint_{C_k} f(z)\,\mathrm{d}z.
\]
\end{dingli}

\begin{proof}
为了把多连通区域化成单连通区域，在外边界与每个内边界之间各作一条互不相交的割线，把区域切开，得到一个单连通区域 $G$。图中的灰色虚线表示的就是这些割线。

在 $G$ 内，$f(z)$ 仍然解析。由单连通区域中的柯西积分定理，
\[
    \oint_{\partial G} f(z)\,\mathrm{d}z = 0.
\]
现在把 $\partial G$ 分成三部分来看：外边界 $C$、内边界 $\Gamma_1,\Gamma_2,\ldots,\Gamma_n$，以及每条割线的两侧边界。

沿同一条割线的两侧，积分经过的是同样一段路径，只是方向相反，因此这两部分积分互相抵消。于是剩下
\[
    \oint_C f(z)\,\mathrm{d}z + \sum_{k=1}^{n} \oint_{\Gamma_k} f(z)\,\mathrm{d}z = 0.
\]
这就证明了第一种写法。

若把每条内边界改成逆时针方向，记为 $C_k$，那么
\[
    \oint_{\Gamma_k} f(z)\,\mathrm{d}z = -\oint_{C_k} f(z)\,\mathrm{d}z.
\]
代回上式，就得到
\[
    \oint_C f(z)\,\mathrm{d}z = \sum_{k=1}^{n} \oint_{C_k} f(z)\,\mathrm{d}z.
\]
\end{proof}

\begin{tishi}[这条定理的意义和用法]
这条定理说明：当区域里出现``洞''以后，外边界上的积分不再必然是零，而是会被这些洞周围的边界积分补出来。也就是说，多连通区域里的积分会真正感受到区域的拓扑结构。

做题时，只要出现``穿孔平面''、``环域''、``把某点围住''这类信息，就不要再机械套用单连通情形的柯西积分定理。更有效的做法通常是：把大曲线缩到围住奇点的小圆上，再去算这些小圆积分。
\end{tishi}

\begin{lizi}{}{}
计算 $\displaystyle\oint_C \frac{1}{z - z_0}\,\mathrm{d}z$，其中 $C$ 是包围 $z_0$ 的简单闭曲线。

\begin{jie}
    以 $z_0$ 为圆心、$r$ 为半径作小圆 $C_r$，取逆时针方向。
    
    由上一定理的等价形式（把内边界改取逆时针方向）：
    \begin{equation}
        \oint_C \frac{1}{z - z_0}\,\mathrm{d}z = \oint_{C_r} \frac{1}{z - z_0}\,\mathrm{d}z
    \end{equation}
    
    在 $C_r$ 上，$z - z_0 = r\mathrm{e}^{\mathrm{i}\theta}$，$\mathrm{d}z = r\mathrm{i}\mathrm{e}^{\mathrm{i}\theta}\,\mathrm{d}\theta$：
    \begin{equation}
        \oint_{C_r} \frac{1}{z - z_0}\,\mathrm{d}z = \int_0^{2\pi} \frac{r\mathrm{i}\mathrm{e}^{\mathrm{i}\theta}}{r\mathrm{e}^{\mathrm{i}\theta}}\,\mathrm{d}\theta = \int_0^{2\pi} \mathrm{i}\,\mathrm{d}\theta = 2\pi\mathrm{i}
    \end{equation}
\end{jie}
\end{lizi}

\begin{tishi}[重要结论]
这个例子的结果非常重要：$\displaystyle\oint_C \frac{1}{z - z_0}\,\mathrm{d}z = 2\pi\mathrm{i}$（$z_0$ 在 $C$ 内部）。这是导出柯西积分公式的基础。
\end{tishi}

\section{柯西积分公式}

\subsection{公式的推导}

\begin{dingli}{柯西积分公式}{}
设 $f(z)$ 在简单闭曲线 $C$ 所围区域 $D$ 内解析，在 $\bar{D} = D \cup C$ 上连续，$z_0$ 是 $D$ 内任意一点，则
\[
    f(z_0) = \frac{1}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{z - z_0}\,\mathrm{d}z
\]
\end{dingli}

图~\ref{fig:cauchy-formula-geometry} 展示了证明里的核心操作：在去掉小圆 $C_r$ 之后得到一个环形区域，再利用多连通区域中的柯西积分定理，把外边界 $C$ 上的积分转移到小圆 $C_r$ 上。

\begin{proof}
考虑积分 $\displaystyle\oint_C \frac{f(z)}{z - z_0}\,\mathrm{d}z$。

以 $z_0$ 为圆心、$r$ 为半径作小圆 $C_r$。由多连通区域的柯西积分定理\index{柯西积分定理}：
\begin{equation}
    \oint_C \frac{f(z)}{z - z_0}\,\mathrm{d}z = \oint_{C_r} \frac{f(z)}{z - z_0}\,\mathrm{d}z
\end{equation}

注意到 $f(z) = f(z_0) + [f(z) - f(z_0)]$，所以
\begin{equation}
    \oint_{C_r} \frac{f(z)}{z - z_0}\,\mathrm{d}z = f(z_0)\oint_{C_r} \frac{1}{z - z_0}\,\mathrm{d}z + \oint_{C_r} \frac{f(z) - f(z_0)}{z - z_0}\,\mathrm{d}z
\end{equation}

第一项等于 $f(z_0) \cdot 2\pi\mathrm{i}$。

对于第二项，由于 $f(z)$ 在 $z_0$ 处连续，对于任意 $\varepsilon > 0$，存在 $\delta > 0$，当 $|z - z_0| < \delta$ 时，$|f(z) - f(z_0)| < \varepsilon$。

取 $r < \delta$，则
\begin{align}
    \left|\oint_{C_r} \frac{f(z) - f(z_0)}{z - z_0}\,\mathrm{d}z\right|
    &\leq \oint_{C_r} \left|\frac{f(z) - f(z_0)}{z - z_0}\right|\,\mathrm{d}s \\
    &\leq \max_{z\in C_r}\left|\frac{f(z)-f(z_0)}{z-z_0}\right|\cdot 2\pi r \\
    &\leq \frac{\varepsilon}{r}\cdot 2\pi r = 2\pi\varepsilon.
\end{align}

这说明第二项的绝对值可以做得任意小，因此它必须等于零。于是
\begin{equation}
    \oint_C \frac{f(z)}{z - z_0}\,\mathrm{d}z = 2\pi\mathrm{i} f(z_0)
\end{equation}
\end{proof}

\begin{wulibj}[物理意义：平均值性质]
柯西积分公式表明：解析函数\index{解析函数}在一点的值等于它在该点周围边界上的``平均值''（加权积分）。这和第二章里调和函数的平均值性质，其实属于同一条直觉：在没有源的二维区域里，内部一点的状态不能脱离边界而任意摆动。

换句话说，边界一旦固定，内部就会受到整体约束。把这种想法放回物理图景里看，它意味着二维势场、稳恒温度场这类无源问题中，内部信息不是独立自由的，而是被边界数据锁定的。这正是解析函数\index{解析函数}``刚性''的一个非常直观的表现。
\end{wulibj}

\begin{tishi}[做题时怎样想到柯西积分公式]
看到
\[
    \oint_C \frac{f(z)}{z-z_0}\,\mathrm{d}z
\]
这种形状时，第一反应就应该是检查两件事：$f(z)$ 是否在 $C$ 及其内部解析，$z_0$ 是否在 $C$ 的内部。两者都满足时，就可以把整个积分一下子化成 $2\pi\mathrm{i}f(z_0)$；若 $z_0$ 不在内部，那么被积函数在曲线内部解析，积分反而直接为零。
\end{tishi}

\begin{tishi}[深入理解：解析函数的``刚性'']
柯西积分公式揭示了解析函数\index{解析函数}的一个惊人性质——让我们来仔细品味一下。

公式告诉我们：只要知道了解析函数\index{解析函数} $f(z)$ 在边界 $C$ 上的值，就能唯一确定它在整个区域内部的值！

这和实变函数完全不同。对于实函数 $y = f(x)$，知道 $f(0) = 1$，你无法确定 $f(0.5)$ 是多少——可能有无数种函数通过 $(0, 1)$ 这个点。但解析函数\index{解析函数}不同：知道了边界上的值，内部就完全确定了。

这就是所谓的``刚性''：解析函数\index{解析函数}就像一块刚性的板，你不能随意弯曲它——一旦固定了边界，整块板就固定了。

更进一步，高阶导数公式告诉我们：解析函数\index{解析函数}在某点的各阶导数也都由边界值决定！这意味着：
\begin{enumerate}
    \item 解析函数\index{解析函数}在一点知道它的值，就知道了它所有导数的值，从而知道了它在整个邻域内的值（泰勒展开）。
    \item 解析函数\index{解析函数}不可能只在某个小区域为零——如果它在一段弧上为零，就必须在整个区域内为零。
    \item 解析函数\index{解析函数}不可能有``局部极值''——最大模原理说 $|f(z)|$ 的最大值只能在边界上达到。
\end{enumerate}

这些性质都源于一个核心：解析函数\index{解析函数}是一个高度``约束''的对象，它的各个部分紧密联系，牵一发而动全身。
\end{tishi}

\section{柯西积分公式的直接推论}

\subsection{高阶导数公式}

柯西积分公式已经告诉我们：边界积分可以恢复内部一点的函数值。一个更进一步的问题自然会出现：如果我们想恢复的不只是函数值，而是这一点附近更细的局部变化信息，例如一阶导数、二阶导数，甚至更高阶导数，该怎么办？

答案是：只要把分母里的幂次提高，边界积分就会对 $z_0$ 附近的行为变得更敏感，从而把更高阶的局部信息提取出来。下面的公式正是在回答这个问题。

\begin{dingli}{高阶导数公式}{}
在柯西积分公式的条件下，$f(z)$ 在 $D$ 内有\keyword{任意阶导数}，且
\[
    f^{(n)}(z_0) = \frac{n!}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{(z - z_0)^{n+1}}\,\mathrm{d}z
\]
\end{dingli}

\label{fig:cauchy-derivative-geometry}
\end{figure}

图~\ref{fig:cauchy-derivative-geometry} 说明了证明里为什么要先限制 $|h|<d/2$：这样 $z_0$ 和 $z_0+h$ 都留在同一条闭曲线 $C$ 的内部，从而可以把两点的函数值都写成同一个边界积分的形式，再去做差商。

\begin{proof}
先证一阶导数公式。由于 $z_0$ 在闭曲线 $C$ 的内部，而 $C$ 是紧曲线，所以 $z_0$ 到 $C$ 的距离
\[
    d=\min_{z\in C}|z-z_0|
\]
是一个正数。取 $|h|<d/2$，则 $z_0+h$ 仍在 $C$ 的内部，并且对 $z\in C$ 有 $z-z_0-h\neq 0$。

分别把柯西积分公式应用于 $z_0$ 和 $z_0+h$，得
\[
    f(z_0+h)=\frac{1}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{z-z_0-h}\,\mathrm{d}z,
    \qquad
    f(z_0)=\frac{1}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{z-z_0}\,\mathrm{d}z.
\]
两式相减并除以 $h$，得到
\[
    \frac{f(z_0+h)-f(z_0)}{h}
    = \frac{1}{2\pi\mathrm{i}}\oint_C
    f(z)\frac{1}{(z-z_0-h)(z-z_0)}\,\mathrm{d}z.
\]
当 $h\to 0$ 时，右端被积函数对 $z\in C$ 一致收敛到
\[
    \frac{f(z)}{(z-z_0)^2},
\]
这一点要稍微说明一下。对任意 $z\in C$，由 $|z-z_0|\ge d$ 以及 $|h|<d/2$ 可知
\[
    |z-z_0-h|\ge |z-z_0|-|h| \ge \frac{d}{2}.
\]
于是
\begin{align}
    \left|
    \frac{f(z)}{(z-z_0-h)(z-z_0)} - \frac{f(z)}{(z-z_0)^2}
    \right|
    &= |f(z)|\left|\frac{h}{(z-z_0-h)(z-z_0)^2}\right| \\
    &\le \max_{\zeta\in C}|f(\zeta)|\cdot \frac{2|h|}{d^3}.
\end{align}
右端与 $z$ 无关，并且当 $h\to 0$ 时趋于零，所以这里确实是一致收敛。于是可以把极限移入积分号内，得到
\[
    f'(z_0)=\frac{1}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{(z-z_0)^2}\,\mathrm{d}z.
\]

接下来记
\[
    I_n(z_0)=\frac{1}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{(z-z_0)^{n+1}}\,\mathrm{d}z.
\]
和上面完全类似，可以对 $I_n(z_0)$ 再求导，得到
\[
    I_n'(z_0)=(n+1)I_{n+1}(z_0).
\]
由于 $f(z_0)=I_0(z_0)$，所以
\[
    f'(z_0)=I_0'(z_0)=I_1(z_0),\qquad
    f''(z_0)=I_1'(z_0)=2I_2(z_0),
\]
继续下去便得到
\[
    f^{(n)}(z_0)=n!I_n(z_0)
    = \frac{n!}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{(z-z_0)^{n+1}}\,\mathrm{d}z.
\]
定理得证。
\end{proof}

\begin{tishi}[惊人结论]
解析函数\index{解析函数}有任意阶导数！这是与实变函数的根本区别。一个实函数可以一阶可导但二阶不可导，但复变函数一旦可导就无穷次可导。
\end{tishi}

若闭路积分里出现
\[
    \frac{f(z)}{(z-z_0)^{n+1}}
\]
这样的结构，而且 $f(z)$ 在曲线及其内部解析，那么最省力的做法通常不是展开分式，而是立刻把它识别成高阶导数公式：先看清 $n$ 是多少，再去算 $f^{(n)}(z_0)$。只有当 $z_0$ 不在曲线内部，或者 $f$ 本身不解析时，这条公式才不能直接使用。

    \end{equation}

    \begin{equation}
        \oint_C \frac{\sin z}{(z - \mathrm{i})^3}\,\mathrm{d}z = \pi\mathrm{i} \cdot (-\mathrm{i}\sinh(1)) = \pi\sinh(1)
    \end{equation}
\end{jie}
\end{lizi}

\subsection{莫雷拉定理}

\begin{dingli}{莫雷拉定理}{}
设 $f(z)$ 在区域 $D$ 内连续，且对于 $D$ 内任意简单闭曲线 $C$，有 $\oint_C f(z)\,\mathrm{d}z = 0$，则 $f(z)$ 在 $D$ 内解析。
\end{dingli}

\label{fig:morera-intuition}
\end{figure}

图~\ref{fig:morera-intuition} 画出了莫雷拉定理背后的直观想法：如果围成区域的小闭路积分都为零，那么把这片区域切成很多小块以后，内部边界会两两抵消，最后就会导向路径无关。

\begin{proof}
固定一点 $z_0\in D$。先说明为什么两点之间的积分与路径无关。设 $C_1,C_2$ 是 $D$ 内两条从 $z_0$ 到 $z$ 的分段光滑曲线。把 $C_2$ 反向后与 $C_1$ 拼起来，就得到一条闭路。若这条闭路有自交点，可以像图~\ref{fig:morera-intuition} 所示那样，把它分解成有限条简单闭曲线；由于条件保证每一条简单闭曲线上的积分都为零，而内部公共边界又会两两抵消，所以最终得到
\[
    \int_{C_1} f(\zeta)\,\mathrm{d}\zeta-\int_{C_2} f(\zeta)\,\mathrm{d}\zeta = 0.
\]
这就说明两点之间的积分与路径无关。于是可以定义
\[
    F(z) = \int_{z_0}^z f(\zeta)\,\mathrm{d}\zeta.
\]

下面证明 $F'(z)=f(z)$。固定任意 $z\in D$。由于 $D$ 是开区域，当 $|h|$ 充分小时，从 $z$ 到 $z+h$ 的线段完全落在 $D$ 内。由路径无关性，
\[
    F(z+h)-F(z)=\int_z^{z+h} f(\zeta)\,\mathrm{d}\zeta.
\]
把线段参数化为
\[
    \zeta=z+th,\qquad 0\le t\le 1,
\]
则
\[
    \mathrm{d}\zeta=h\,\mathrm{d}t,
\]
从而
\[
    \frac{F(z+h)-F(z)}{h}
    = \int_0^1 f(z+th)\,\mathrm{d}t.
\]
由于 $f$ 在 $z$ 处连续，对任意 $\varepsilon>0$，存在 $\eta>0$，当 $|w-z|<\eta$ 时就有
\[
    |f(w)-f(z)|<\varepsilon.
\]
当 $|h|<\eta$ 时，对所有 $t\in[0,1]$ 都有
\[
    |z+th-z|=t|h|\le |h|<\eta,
\]
所以整条小线段上的函数值都会同时靠近 $f(z)$。于是
\begin{align}
    \left|\int_0^1 f(z+th)\,\mathrm{d}t-\int_0^1 f(z)\,\mathrm{d}t\right|
    &\le \int_0^1 |f(z+th)-f(z)|\,\mathrm{d}t \\
    &\le \int_0^1 \varepsilon\,\mathrm{d}t = \varepsilon.
\end{align}
因此
\[
    \lim_{h\to 0}\frac{F(z+h)-F(z)}{h}
    = \int_0^1 f(z)\,\mathrm{d}t
    = f(z).
\]
这说明 $F'(z)=f(z)$，所以 $F$ 在 $D$ 内解析，而 $f=F'$。由上一节得到的高阶导数公式可知，解析函数的导数仍然解析，因此 $f(z)$ 在 $D$ 内解析。
\end{proof}

\begin{tishi}[柯西积分定理的逆定理]
    莫雷拉\index{莫雷拉@莫雷拉（Morera）}（Morera）定理是柯西积分定理\index{柯西积分定理}的逆定理。它的意义在于：``闭路积分为零''不只是解析函数的结果，在连续性已经知道的前提下，它反过来还可以成为判断解析性的工具。

做题时，如果题目已经给出函数连续，而且又告诉你对任意闭曲线都有
\[
    \oint_C f(z)\,\mathrm{d}z=0,
\]
那就不必再去硬验柯西--黎曼条件，而应直接想到莫雷拉定理。两者结合，我们得到：
\begin{equation}
    f \text{ 解析} \quad \Longleftrightarrow \quad \oint_C f(z)\,\mathrm{d}z = 0 \text{（对任意闭曲线）}
\end{equation}
\end{tishi}

读这张图时，可以把自己想象成正在复盘一整章课：先学会怎样沿曲线真正去算复积分，并掌握最基本的估值工具；再通过原函数把积分和端点联系起来；接着用柯西积分定理解释为什么解析函数在单连通区域里会突然变得路径无关；而一旦区域里有洞，积分行为又会立刻变样。最后，所有这些内容都会汇聚到柯西积分公式那里，并继续长出高阶导数公式、解析函数的刚性，以及下一章的泰勒展开。只要这条线是顺的，本章就算真正学通了。

\begin{zhongyao}[本章主要结论]
\raggedright
\begin{enumerate}
    \item \textbf{复积分首先是沿曲线的积分：}复积分不是把实积分里的 $\mathrm{d}x$ 机械地换成 $\mathrm{d}z$，而是要沿一条有向曲线作和，并通过参数化
    \[
        z=z(t)
    \]
    把它化成实变量积分来计算。与此同时，积分估值不等式
    \[
        \left|\int_C f(z)\,\mathrm{d}z\right|
        \le \int_C |f(z)|\,\mathrm{d}s
    \]
    又提供了后面证明里最常用的控制工具。真正做题时，先把路径参数化清楚，再看是否需要估值，往往就已经完成了一半。

    \item \textbf{有原函数时，积分只由端点决定：}若
    \[
        F'(z)=f(z),
    \]
    那么
    \[
        \int_C f(z)\,\mathrm{d}z=F(z_2)-F(z_1).
    \]
    这就是复积分里的牛顿--莱布尼茨公式。它告诉我们：一旦原函数存在，积分立刻与路径无关，闭路积分也立刻为零。

    \item \textbf{柯西积分定理是整章的枢纽：}在单连通区域里，解析函数满足
    \[
        \oint_C f(z)\,\mathrm{d}z=0.
    \]
    这不是一个孤立结论，而是后面一整串性质的源头：闭路积分为零、路径无关、原函数存在，都从这里生长出来。也正因为如此，柯西积分定理不是一条普通公式，而是复积分理论真正开始变得``整齐''的地方。

    \item \textbf{区域里有没有洞会直接改变积分行为：}单连通条件不是可以随手丢掉的小限制。像
    \[
        \frac{1}{z-z_0}
    \]
    这样的函数，只要闭曲线把 $z_0$ 围住，就有
    \[
        \oint_C \frac{1}{z-z_0}\,\mathrm{d}z=2\pi\mathrm{i}.
    \]
    这说明多连通区域中的积分会真正感受到区域拓扑结构的影响。

    \item \textbf{柯西积分公式把边界和内部直接连了起来：}若 $f$ 在边界内部解析，那么
    \[
        f(z_0)=\frac{1}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{z-z_0}\,\mathrm{d}z.
    \]
    这句话的意义非常深：它说明解析函数在内部一点的值，不是孤立的，而是被边界积分完全控制。这正是解析函数``刚性''的最早表现。

    \item \textbf{高阶导数也能由边界积分表示：}在柯西积分公式基础上，可以得到
    \[
        f^{(n)}(z_0)=\frac{n!}{2\pi\mathrm{i}}\oint_C \frac{f(z)}{(z-z_0)^{n+1}}\,\mathrm{d}z.
    \]
    这说明解析函数一旦可导，就不只是``再可导一次''，而是会自动拥有任意阶导数。下一章的泰勒展开，正是从这里继续展开的。

    \item \textbf{莫雷拉定理给出反方向判据：}若连续函数对任意闭曲线的积分都为零，那么它反过来也解析。也就是说，闭路积分为零不只是解析函数的结果，还可以在连续性条件下成为解析性的判据。
\end{enumerate}
\end{zhongyao}

\begin{tishi}[做题时的判断思路]
\begin{enumerate}
    \item \textbf{直接算积分：}若题目只给出一条具体路径，优先先做参数化；直线段、圆弧、圆周通常最容易下手。不要一看见复积分就急着找大定理，很多题只是参数化计算题。

    \item \textbf{判断是否路径无关：}若题目问积分是否只和端点有关，先想两个问题：函数有没有原函数？或者它是不是在单连通区域内解析？这两条线常常是同一个问题的不同入口。

    \item \textbf{闭路积分是否为零：}看到闭曲线时，不要机械套用柯西积分定理，一定先检查函数是否在曲线及其内部解析，以及区域里有没有洞。单连通条件和``内部没有奇点''这两件事，缺一不可。

    \item \textbf{遇到 $\dfrac{f(z)}{z-z_0}$：}若 $f$ 在闭曲线内部解析，且 $z_0$ 在曲线内部，优先想到柯西积分公式；若 $z_0$ 在曲线外部，积分往往直接为零。

    \item \textbf{遇到高阶极形式：}若分母是
    \[
        (z-z_0)^{n+1},
    \]
    就要主动切换到高阶导数公式，而不是先做繁琐的代数展开。

    \item \textbf{多连通区域题：}只要题目里出现``去心平面''、``环域''或``绕原点一圈''这类信息，就要立刻提高警惕：这里最关键的往往不是函数本身多复杂，而是闭曲线有没有把洞或奇点围进去。

    \item \textbf{反向判定解析：}若题目已知函数连续，并且对任意闭曲线积分都为零，就要想到莫雷拉定理；这类题不是继续硬算积分，而是要把积分信息翻译回解析性。
\end{enumerate}
\end{tishi}

\begin{jinshi}[本章最容易混淆的地方]
\begin{enumerate}
    \item 不要把``有原函数 $\Rightarrow$ 路径无关''和``任意函数都路径无关''混为一谈；对一般连续函数，积分本来就会依赖路径。
    \item 不要忽略方向。把曲线反向以后，积分会变号；多连通区域里外边界和内边界的取向也不同。
    \item 不要把柯西积分定理的单连通条件当成装饰。区域里有洞时，闭路积分完全可能不为零。
    \item 不要在函数内部有奇点时仍然机械套用柯西积分定理或柯西积分公式。
    \item 不要把
    \[
        \oint_C \frac{1}{z-z_0}\,\mathrm{d}z=2\pi\mathrm{i}
    \]
    误当成对所有闭曲线都成立；关键是 $C$ 是否把 $z_0$ 围住，以及方向是否为正向。
    \item 不要把莫雷拉定理的使用条件忘掉；它需要先有连续性，不能只凭``闭路积分像是零''就直接下结论。
\end{enumerate}
\end{jinshi}

\begin{tishi}[和后续内容的联系]
这一章和第二章的关系非常紧：第二章告诉我们什么是解析函数，以及为什么它具有强结构；第三章则第一次把这种结构放到积分层面，说明解析性会怎样改变闭路积分、原函数和边界信息之间的关系。

下一章进入泰勒展开、解析函数的整体性质与刚性时，真正起支撑作用的正是本章的柯西积分公式和高阶导数公式；再往后到第五章留数理论时，本章里反复出现的典型积分
\[
    \oint_C \frac{1}{z-z_0}\,\mathrm{d}z=2\pi\mathrm{i}
\]
又会被推广成更系统的计算方法。也就是说，第三章既是复积分的起点，也是后面展开理论和留数理论的桥梁。
\end{tishi}

\section{习题}

\textbf{基础题}

\begin{enumerate}
    \item 计算下列积分：
    \begin{enumerate}
        \item $\displaystyle\int_C z^2\,\mathrm{d}z$，$C$ 是从 $0$ 到 $1 + \mathrm{i}$ 的直线段
        \item 分别沿两条路径计算 $\displaystyle\int_C \overline{z}\,\mathrm{d}z$：一路径为从 $0$ 到 $1+\mathrm{i}$ 的直线段，另一路径先沿实轴从 $0$ 到 $1$，再沿竖直线段从 $1$ 到 $1+\mathrm{i}$
    \end{enumerate}
    
    \item 利用柯西积分公式计算：
    \begin{enumerate}
        \item $\displaystyle\oint_C \frac{\cos z}{z - \pi}\,\mathrm{d}z$，$C$ 是 $|z| = 4$
        \item $\displaystyle\oint_C \frac{\mathrm{e}^z}{z^2}\,\mathrm{d}z$，$C$ 是 $|z| = 1$
    \end{enumerate}
\end{enumerate}

\textbf{综合题}

\begin{enumerate}[resume]
    \item 设 $f(z)$ 在区域 $D$ 内有原函数 $F(z)$。证明：$f(z)$ 在 $D$ 内的积分与路径无关，并说明为什么任意闭曲线上的积分都为零。
    
    \item 计算 $\displaystyle\oint_C \frac{z}{(z - 1)(z - 2)}\,\mathrm{d}z$，其中 $C$ 分别为：
    \begin{enumerate}
        \item $|z| = 1.5$
        \item $|z| = 3$
    \end{enumerate}
    
    \item 说明为什么 $\displaystyle \frac{1}{z}$ 在穿孔平面 $\mathbb{C}\setminus\{0\}$ 上不可能有单值原函数。
\end{enumerate}

\textbf{挑战题}

\begin{enumerate}[resume]
    \item 用参数化方法直接证明：若 $C$ 是圆周 $|z-z_0|=r$ 的逆时针方向，则
    \[
        \oint_C \frac{1}{z-z_0}\,\mathrm{d}z = 2\pi\mathrm{i}.
    \]
    
    \item 设 $f(z)$ 在区域 $D$ 内连续，且对 $D$ 内任意闭曲线 $C$ 都有
    \[
        \oint_C f(z)\,\mathrm{d}z = 0.
    \]
    取定 $z_0\in D$，定义
    \[
        F(z)=\int_{z_0}^{z} f(\zeta)\,\mathrm{d}\zeta.
    \]
    试证明 $F'(z)=f(z)$，并说明这与莫雷拉定理有什么关系。
\end{enumerate}
